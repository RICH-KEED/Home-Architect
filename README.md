# Dream to Home

A production Next.js scroll-driven storytelling site where users scrub through construction frames and transition into a final orbit video.

## Architecture

```txt
app/          Next.js app router entry points and global styles
components/   UI and animation components
hooks/        Reusable client-side animation/loading hooks
lib/          Constants and pure utilities
types/        Shared TypeScript types
styles/       Global design/style helpers
public/       Runtime assets served by Next.js
```
